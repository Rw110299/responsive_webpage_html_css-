# 📜 Código de Conduta

## 1️⃣ Nosso Compromisso
Promover um ambiente aberto e acolhedor, garantindo que todos, independentemente de nível de experiência, gênero, identidade, orientação sexual, deficiência ou religião, possam contribuir de forma respeitosa.

## 2️⃣ Comportamentos Esperados
✅ Uso de linguagem respeitosa e inclusiva.  
✅ Aceitar críticas construtivas.  
✅ Focar no que é melhor para a comunidade.  
✅ Mostrar empatia com outros participantes.

## 3️⃣ Comportamentos Inaceitáveis
🚫 Uso de linguagem discriminatória ou ofensiva.  
🚫 Ataques pessoais ou políticos.  
🚫 Comportamento intimidador ou de assédio.  
🚫 Publicação de informações pessoais de outros sem permissão.

## 4️⃣ Aplicação
Violadores deste código poderão ter ações corretivas, incluindo banimento temporário ou permanente dos espaços de colaboração.

## 5️⃣ Contato
Para relatar qualquer violação deste código de conduta, envie um e-mail para **seuemail@dominio.com** ou abra uma issue privada no GitHub.
